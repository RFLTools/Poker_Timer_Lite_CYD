⚠️ Before You Start

  • Required Hardware: ESP32-2432S028 (CYD - Cheap Yellow Display)
  • Browser: Chrome, Edge, or Opera (Web Serial API support required)
  • Cable: USB cable to connect your device
  • Driver: CP210x or CH340 USB driver may be needed

🚀 Install Firmware

Click the button below to connect your ESP32 device and install the firmware.

This will erase the existing firmware and install the Poker Timer application

📋 Installation Steps

1
Connect Device
Connect your ESP32-2432S028 to your computer using a USB cable
2
Click Install Button
Click "INSTALL POKER TIMER" and select your device from the popup
3
Wait for Installation
The firmware will be uploaded automatically (takes ~1-2 minutes)
4
Configure & Play
Tap the gear icon on the device to configure WiFi and customize tournament settings

❓ Troubleshooting

Port not showing up?

  • Install the CP210x or CH340 USB driver for your operating system
  • Try a different USB cable (some cables are power-only)
  • Try a different USB port on your computer

Installation failed?

  • Hold the BOOT button while connecting the USB cable
  • Try again with a different browser (Chrome or Edge)
  • Ensure no other program is using the serial port
